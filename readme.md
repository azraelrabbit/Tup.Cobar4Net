Tup.Cobar4Net
====================

Cobar 是提供关系型数据库 (MySQL) 分布式服务的中间件, 它可以让传统的数据库得到良好的线性扩展, 并看上去还是一个数据库, 对应用保持透明.

本项目为 Alibaba [Cobar](https://github.com/alibaba/cobar/) .NET 实现版.

#目前功能:
	MySQL Sql Parser
	Route
	Config
